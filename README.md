# CodeAlpha_BasicChatBot
A simple rule-based chatbot in Python that responds to basic user inputs using conditional statements and loops.
#  Basic Chatbot in Python

This is a simple rule-based chatbot developed using Python as part of my internship project.

# Project Description
The chatbot interacts with users by responding to predefined inputs such as greetings and simple questions. It demonstrates basic conversational logic using conditional statements.

# Features
- Responds to basic user inputs like:
  - "hello"
  - "how are you"
  - "bye"
- Simple and easy-to-understand logic
- Continuous interaction using a loop until the user exits

# Technologies Used
- Python

#  Concepts Used
- if-elif statements
- loops
- functions
- user input/output
